# 📝 Task Manager

Aplicación web simple para gestionar tareas desarrollada con **React, Node.js y Express**.
Permite crear, visualizar, completar y eliminar tareas.

---

## 🚀 Funcionalidades

- Crear nuevas tareas
- Ver la lista de tareas
- Marcar tareas como completadas
- Eliminar tareas
- Interfaz moderna y simple

---

## 🛠 Tecnologías utilizadas

### Frontend

- React
- Vite
- TailwindCSS

### Backend

- Node.js
- Express

---

## 📦 Instalación y ejecución

### 1️⃣ Clonar el repositorio

```bash
git clone <url-del-repositorio>
cd forit-task-app
```

---

### 2️⃣ Ejecutar el backend

```bash
cd backend
npm install
npm run dev
```

El backend se ejecutará en:

```
http://localhost:3000
```

---

### 3️⃣ Ejecutar el frontend

En otra terminal:

```bash
cd frontend
npm install
npm run dev
```

El frontend se ejecutará en:

```
http://localhost:5173
```
